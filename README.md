🌸 Aura Handmade – E-Commerce Web Application

Tagline:
Handmade Happiness for Every Occasion — Crafted with Love, Made for You.

Aura Handmade is a full-stack e-commerce web application designed to manage and sell custom handmade products online.
The platform provides a seamless shopping experience for users and a powerful management system for administrators.

The application includes two main modules:

👤 User Module

🛠 Admin Module

It also integrates cart, wishlist, order management, and online payment functionality.

🏗 System Architecture

The application follows a Client–Server Architecture.

Frontend

React.js (Single Page Application)

Backend

Node.js

Express.js

Mongoose

Database

MongoDB

Authentication

JWT (JSON Web Token) Authentication

Role-Based Access Control

Payment Integration

Stripe Payment Gateway

👥 Application Modules
1️⃣ User Module

The User Module allows customers to explore and purchase handmade products.

Features

User Registration & Login (JWT Authentication)

Browse Products

Search & Filter Products

View Product Details

Add Products to Cart

Add Products to Wishlist

Remove Products from Cart / Wishlist

Place Orders

Online Payment Integration

View Order History

User Flow

User registers or logs into the system

Browses available products

Adds products to Cart or Wishlist

Proceeds to Checkout

Completes Online Payment

Order details are stored in the database

2️⃣ Admin Module

The Admin Module allows administrators to manage products, users, and orders.

Features

Secure Admin Login

Add Products

Update Products

Delete Products

Manage Product Categories

View All Orders

Update Order Status

Manage Users (Activate / Deactivate)

Admin Flow

Admin logs into the admin dashboard

Performs CRUD operations on products

Monitors and manages orders

Updates order status

Manages user accounts

🔐 Authentication & Security

The application implements several security practices:

Password Hashing using bcrypt

JWT Token-Based Authentication

Role-Based Access Control (Admin/User)

Protected Routes using Middleware

📦 Frontend Packages Used

The frontend uses several libraries to improve functionality and UI.

react-router-dom → Routing

Material UI → UI components

Swiper → Carousel / Product sliders

Context API → Global state management (Auth, Modals)

Axios → API communication

react-oauth/google → Google Login integration

jwt-decode → Decode Google credentials

Iconify → Icons

⚙️ Backend Technologies

Node.js

Express.js

MongoDB

Mongoose

JWT Authentication

Stripe Payment Integration

Full E-Commerce functionality

Shopping Cart & Wishlist
